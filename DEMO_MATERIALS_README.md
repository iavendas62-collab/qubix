## üé¨ Quick Start Guide

### For Video Production Team

**Step 1: Review Materials (1 hour)**
1. Read DEMO_VIDEO_SCRIPT.md
2. Review DEMO_SHOT_LIST.md
3. Understand DEMO_PRODUCTION_GUIDE.md

**Step 2: Setup (2 hours)**
1. Install recording software (OBS Studio)
2. Install editing software (DaVinci Resolve)
3. Set up microphone and recording space
4. Prepare QUBIX platform (backend + frontend running)
5. Create test accounts and data

**Step 3: Record Narration (2 hours)**
1. Practice reading script
2. Record in quiet environment
3. Edit and clean up audio
4. Export as WAV file

**Step 4: Record Screen Captures (4 hours)**
1. Follow DEMO_SHOT_LIST.md
2. Record each shot 2-3 times
3. Organize files by shot number
4. Backup all recordings

**Step 5: Edit Video (8 hours)**
1. Import all media
2. Create rough cut
3. Add graphics and effects
4. Color grade footage
5. Mix audio
6. Add captions (use DEMO_CAPTIONS.srt)
7. Final review

**Step 6: Export and Distribute (2 hours)**
1. Export in 1080p HD
2. Create thumbnail
3. Upload to YouTube/Vimeo
4. Write description
5. Share with team

**Total Time: ~20 hours**

## üé• Recording Tips

### Do's ‚úÖ
- **Practice** each shot before recording
- **Move slowly** - give viewers time to see
- **Record multiple takes** - choose best in editing
- **Use smooth transitions** - no jarring cuts
- **Highlight key elements** - zoom, arrows, boxes
- **Show real data** - actual blockchain transactions
- **Be enthusiastic** - passion is contagious

### Don'ts ‚ùå
- **Don't rush** - viewers need time to absorb
- **Don't use fake data** - authenticity matters
- **Don't over-edit** - keep it natural
- **Don't use flashy effects** - stay professional
- **Don't forget captions** - accessibility is important
- **Don't skip testing** - verify everything works

## üÜò Troubleshooting

### Common Issues

**Problem: Audio is too quiet**
- Solution: Normalize audio to -6dB peak
- Use compression to even out levels
- Boost in post-production

**Problem: Video is choppy**
- Solution: Lower recording frame rate to 30fps
- Close other applications
- Record in shorter segments

**Problem: File size too large**
- Solution: Use H.264 codec
- Lower bitrate to 6-8 Mbps
- Compress in HandBrake

**Problem: Colors look washed out**
- Solution: Adjust saturation (+10-15%)
- Increase contrast slightly
- Use color grading tools

**Problem: Captions out of sync**
- Solution: Adjust timing in SRT file
- Re-import captions
- Manually adjust in editor
