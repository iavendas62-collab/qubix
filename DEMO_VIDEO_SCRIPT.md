## Technical Recording Notes

### Screen Recording Settings
- Resolution: 1920x1080 (1080p)
- Frame rate: 60fps
- Bitrate: 8-10 Mbps
- Format: MP4 (H.264 codec)

### Audio Settings
- Narration: Clear voice, moderate pace
- Background music: Royalty-free, upbeat tech music
- Music volume: -20dB (background level)
- Voice volume: 0dB (foreground level)

### Editing Guidelines
1. Use smooth transitions between segments (0.5s fade)
2. Add zoom effects on important UI elements
3. Highlight mouse cursor for key actions
4. Add motion graphics for statistics
5. Include captions for all narration
6. Add subtle animations for on-screen text
7. Color grade for professional look

### Caption Timing
- Display captions 0.5s before narration
- Keep captions on screen for 2-3 seconds
- Use large, readable font (Arial Bold, 48pt)
- White text with black outline for visibility

### Music Suggestions (Royalty-Free)
- "Tech Innovation" by Bensound
- "Digital Future" by Audionautix
- "Upbeat Corporate" by AShamaluevMusic
- "Technology" by Scott Holmes Music

### Export Settings
- Format: MP4
- Codec: H.264
- Resolution: 1920x1080
- Frame rate: 30fps (for final export)
- Bitrate: 8 Mbps
- Audio: AAC, 192 kbps, 48kHz
